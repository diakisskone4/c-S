1 dir : Affiche la liste des fichiers et dossiers dans le répertoire en cours.
 2 cd : Change le répertoire actif dans lequel vous vous trouvez.
 3 mkdir : Crée un nouveau dossier.
 4 type : Affiche le contenu d'un fichier texte.
 5 del : Supprime un fichier.
 6 ipconfig : Affiche les informations sur la configuration réseau de votre ordinateur.
 7 ping : Vérifie la connectivité réseau vers une adresse IP ou un nom de domaine.
 8 tasklist : Affiche la liste des processus en cours d'exécution sur votre ordinateur.
Voici comment utiliser ces commandes :
 1 Pour utiliser la commande dir, ouvrez une fenêtre CMD et tapez dir suivi de la touche Entrée. La liste des fichiers et dossiers dans le répertoire actif s'affichera.
 2 Pour utiliser la commande cd, tapez cd suivi du chemin d'accès du répertoire dans lequel vous souhaitez vous déplacer. Par exemple, pour aller dans le dossier "Documents", tapez cd Documents et appuyez sur Entrée.
 3 Pour créer un nouveau dossier, tapez mkdir suivi du nom du dossier que vous souhaitez créer. Par exemple, pour créer un dossier appelé "Nouveau dossier", tapez mkdir "Nouveau dossier" et appuyez sur Entrée.
 4 Pour afficher le contenu d'un fichier texte, tapez type suivi du chemin d'accès du fichier. Par exemple, pour afficher le contenu d'un fichier appelé "monfichier.txt" qui se trouve dans le dossier "Documents", tapez type C:\Users\VotreNom\Documents\monfichier.txt.
 5 Pour supprimer un fichier, tapez del suivi du chemin d'accès du fichier. Par exemple, pour supprimer un fichier appelé "monfichier.txt" qui se trouve dans le dossier "Documents", tapez del C:\Users\VotreNom\Documents\monfichier.txt.
 6 Pour afficher les informations de configuration réseau, tapez ipconfig et appuyez sur Entrée.
 7 Pour vérifier la connectivité réseau vers une adresse IP ou un nom de domaine, tapez ping suivi de l'adresse IP ou du nom de domaine. Par exemple, pour vérifier la connectivité vers le site Web Google, tapez ping www.google.com et appuyez sur Entrée.
 8 Pour afficher la liste des processus en cours d'exécution sur votre ordinateur, tapez tasklist et appuyez sur Entrée.
